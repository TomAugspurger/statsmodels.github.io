statsmodels.stats.contingency_tables.Table2x2.log_riskratio_confint¶

Table2x2.log_riskratio_confint(alpha=0.05, method='normal')[source]¶

A confidence interval for the log risk ratio.

Parameters:

alpha : float

1 - alpha is the nominal coverage probability of the confidence interval.

method : string

The method for producing the confidence interval. Currently must be ‘normal’ which uses the normal approximation.
